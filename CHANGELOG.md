# Changelog

## v0.3.4 - 2026-06-24

- All-done badge now sits below the mascot as a two-line card instead of overlapping the sprite.
- Complete hero adds floating seed sprites, pulse rings, badge rise animation, and a warm radial glow.

## v0.3.3 - 2026-06-24

- Today hero uses a consistent two-column grid so copy and mascot stay aligned on every screen size.
- Mascot stage is centered in its column; floating chips no longer overflow outside the hero bounds.

## v0.3.2 - 2026-06-24

- Day-complete celebration now shows a large jumping mascot centered on screen with radial burst rings and seed explosion.
- Hero mascot stays centered with a continuous happy hop while all today's tasks are done.

## v0.3.1 - 2026-06-24

- Replaced emoji with Lucide icons and mascot sprites across the app.
- Goal tasks now store plain titles; Wallet and Scale icons render in the UI.
- Celebration burst uses sprout mascot images instead of emoji particles.

## v0.3.0 - 2026-06-24

- Added Goals tab with financial and weight tracking, linked tasks, and per-goal dashboards.
- Aligned Dashboard goal section with Goals — read-only summary with link to full management.
- Financial goals now track daily income and expense with running balance and monthly charts.
- Updated day editor and goal progress to use the shared goal records model.

## v0.2.0 - 2026-06-21

- Added app version details to Settings.
- Added `version.json` as the single release metadata source.
- Added changelog notes for release tracking.

## v0.1.0

- Initial Sprout Planner release.
